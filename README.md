# 🎓 公务员考试智能生成系统 (GWY Exam Generator)\n\n一个基于AI的公考真题分析 + 模拟题自动生成系统\n\n## ✨ 核心特性\n\n- 📊 **智能规律分析**：自动识别出题主题、难度趋势、知识点复现率\n- 🤖 **AI题目生成**：基于规律自动生成独创题目\n- ✅ **质量验证**：多维度评估生成题的质量\n- 🌐 **Web UI**：友好的网页界面\n- 🔗 **GitHub集成**：直接从真题仓库提取数据\n\n## 🚀 快速开始\n\n### 环境要求\n- Python >= 3.9\n- Node.js >= 14 (可选，用于前端)\n\n### 后端启动\n```bash\n# 1. 克隆项目\ngit clone https://github.com/liaopeiyan/gwy-exam-generator.git\ncd gwy-exam-generator\n\n# 2. 创建虚拟环境\npython -m venv venv\nsource venv/bin/activate  # Linux/Mac\n# 或\nvenv\\Scripts\\activate  # Windows\n\n# 3. 安装依赖\npip install -r requirements.txt\n\n# 4. 配置环境\ncp .env.example .env\n\n# 5. 启动后端\ncd backend\npython main.py\n```\n\n访问: http://localhost:5000/api/status\n\n### 前端启动 (可选)\n```bash\ncd frontend\nnpm install\nnpm run dev\n```\n\n访问: http://localhost:3000\n\n## 📁 项目结构\n\n```\ngwy-exam-generator/\n├── backend/\n│   ├── main.py              # 启动文件\n│   ├── app.py               # Flask应用\n│   ├── config.py            # 配置\n│   ├── agents/              # AI智能体\n│   │   ├── analysis_agent.py      # 分析agent\n│   │   ├── generation_agent.py    # 生成agent\n│   │   └── validation_agent.py    # 验证agent\n│   ├── routes/              # API路由\n│   │   └── api.py\n│   ├── tools/               # 工具函数\n│   │   └── data_preprocessor.py\n│   ├── data/                # 数据目录\n│   └── logs/                # 日志\n├── frontend/                # Web UI (Vue 3)\n├── reports/                 # 输出报告\n├── requirements.txt         # Python依赖\n├── .env.example             # 环境变量模板\n└── README.md\n```\n\n## 🔧 API 接口\n\n### 1️⃣ 健康检查\n```bash\nGET /api/status\n```\n\n### 2️⃣ 分析真题\n```bash\nPOST /api/analyze\n\nResponse:\n{\n  \"status\": \"success\",\n  \"message\": \"Analyzed 150 exams\",\n  \"patterns\": {\n    \"theme_distribution\": {...},\n    \"type_distribution\": {...},\n    \"difficulty_distribution\": {...}\n  }\n}\n```\n\n### 3️⃣ 生成题目\n```bash\nPOST /api/generate\n\nBody:\n{\n  \"num_questions\": 100,\n  \"difficulty_distribution\": {\n    \"easy\": 0.3,\n    \"medium\": 0.4,\n    \"hard\": 0.3\n  },\n  \"question_types\": [\"常识\", \"言语\", \"逻辑\", \"数学\", \"资料分析\"]\n}\n\nResponse:\n{\n  \"status\": \"success\",\n  \"message\": \"Generated 100 questions\",\n  \"questions_count\": 100\n}\n```\n\n### 4️⃣ 验证质量\n```bash\nPOST /api/validate\n\nResponse:\n{\n  \"status\": \"success\",\n  \"validation_result\": {\n    \"total_questions\": 100,\n    \"average_quality_score\": 78.5,\n    \"pass_rate\": 85.0,\n    \"quality_distribution\": {\n      \"excellent\": 25,\n      \"good\": 40,\n      \"medium\": 30,\n      \"poor\": 5\n    }\n  }\n}\n```\n\n## 📊 工作流程\n\n```\n历年真题 (GitHub: Yaoyuan-Zhang319/AdministrativeAptitudeTest)\n   ↓\n数据解析 → 结构化JSON\n   ↓\n规律分析 → 提取主题、题型、难度分布\n   ↓\n知识库构建 → 建立关键词和概念映射\n   ↓\n题目生成 → 按规律自动生成新题\n   ↓\n质量验证 → 多维度评估\n   ↓\n📈 最终报告\n```\n\n## 🤖 AI模型配置\n\n目前支持免费模型：\n\n### Ollama (推荐)\n```bash\n# 安装 Ollama\n# https://ollama.ai\n\n# 下载模型\nollama pull mistral\n# 或\nollama pull neural-chat\n\n# 启动服务\nollama serve\n```\n\n配置 `.env`：\n```\nAI_PROVIDER=ollama\nOLLAMA_BASE_URL=http://localhost:11434\nOLLAMA_MODEL=mistral\n```\n\n## 📈 项目进度\n\n- [x]
